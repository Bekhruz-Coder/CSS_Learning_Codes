Example for "em" unit.
If elemet does not have a font-size for the "em",
it will use font-size of its parent. If also parent doesn't have one,
it use font-size of its ancestor.
If element do not have a font-size property,
"em" will use brauser's default font-size, which is usually 16px.
When the element is nested.
